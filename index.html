Objetivo

O AGES Compiler é um compilador desenvolvido para facilitar a programação através de características e sintaxe em português. O objetivo principal é criar uma linguagem de programação própria com sintaxe totalmente em português, simplificando o aprendizado de conceitos de compiladores para iniciantes falantes da língua portuguesa.

Como Foi Feito

Etapas do Compilador

O projeto foi desenvolvido em três etapas principais:

  1. Lexer (lexer.js): Realiza a análise léxica do código, convertendo o arquivo de texto em uma sequência de tokens (palavras-chave, identificadores, operadores, etc.)
  2. Parser (parser.js): Realiza a análise sintática dos tokens, criando uma Árvore de Sintaxe Abstrata (AST) que representa a estrutura do programa.
  3. Interpretador (interpreter.js): Executa a AST gerada pelo parser, interpretando e rodando o código Ages.

Integração

O compilador (compiler.js) orquestra todo o processo, integrando lexer, parser e interpretador em um fluxo único. A interface de linha de comando (bin/agesCompile) permite executar arquivos .ages diretamente do terminal.

Tecnologias utilizadas: JavaScript (Node.js), análise léxica e sintática, AST (Árvore de Sintaxe Abstrata).

Linguagem AGES

Características

A linguagem AGES suporta:

  • Variáveis: variavel nomeDaVariavel = valor
  • Funções: funcao nomeFuncao(parametros) { ... }
  • Condicionais: se (condicao) { ... } senao { ... }
  • Loops: enquanto (condicao) { ... }
  • Saída: escreva(valor)
  • Retorno: retorne valor
  • Tipos: Números, Strings, Booleanos, Null

Exemplo Prático

variavel nome = "Ages Lang" variavel numero = 7 escreva(numero) se numero == 2 { escreva(nome) }

Como Usar

  1. Instalar as dependências: npm install
  2. Criar um arquivo com extensão .ages
  3. Compilar e executar: agesCompile seu_arquivo.ages

Conclusão

O AGES Compiler demonstra a implementação completa de um compilador funcional, desde a análise léxica até a interpretação. O projeto cumpre o objetivo de facilitar a programação em português e serve como ferramenta educacional para aprender conceitos fundamentais de design de linguagens de programação.
